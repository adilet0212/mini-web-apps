# Multiplication Table Generator

An interactive web app that generates a multiplication table based on user-defined ranges.

## Live Demo
https://adilet0212.github.io/mini-web-apps/multiplication-table/

## Features
- Enter row and column values  
- Instantly generate a formatted table  
- Simple validation for input values  

## Tech Stack
HTML, CSS, JavaScript

## Run Locally
1. Clone this repository  
2. Open the folder `multiplication-table`  
3. Open `index.html` in your browser
