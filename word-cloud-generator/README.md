# Word Cloud Generator

Generate a word cloud from input links (websites containing some articles, etc.) with styling and frequency-based word sizing.

## Live Demo
https://adilet0212.github.io/mini-web-apps/word-cloud-generator/

## Features
- Input a link containing an article text and instantly visualize it as a word cloud  
- Frequency-based font sizing
- Simple and responsive UI

## Tech Stack
HTML, CSS, JavaScript

## Run Locally
1. Clone this repository  
2. Open the folder `word-cloud-generator`  
3. Open `index.html` in your browser
